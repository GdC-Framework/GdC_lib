# GdC_lib

Librairie de fonctions pour la communauté Arma 3 Grèce de Canards

Version de compilation avec pboproject, pour compiler :

1. Monter le P drive avec les dossiers du jeu Arma 3 extrait (A3 tools) +
   extraction des PBO du jeu.
1. Dossier GDC_lib à la racine du P drive, copier les addons à build dedans ou
   directement le dépot git.
1. Lancer pboproject et build avec les options +C et -B. Attention par défaut,
   pboproject exlut les fichiers .hpp, or ceux-ci doivent être inclus dans le
   PBO.
